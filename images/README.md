# Images Directory

Store all image assets here for easy management.

## Recommended Files

- **profile.jpg** - Your profile/headshot image (used in hero section)
- **logo.png** - Any custom logos
- Research-related images or diagrams

## Best Practices

1. **Naming**: Use descriptive, lowercase names with hyphens
   - ✅ Good: `profile-photo.jpg`, `research-diagram.png`
   - ❌ Bad: `photo1.jpg`, `DIAGRAM.PNG`

2. **Formats**: 
   - JPG for photos (smaller file size)
   - PNG for graphics/logos (transparent background)
   - SVG for icons (scalable)

3. **Optimization**: 
   - Compress images before uploading
   - Recommended size for profile: 180x180px (already mentioned in CSS)

## Usage in HTML

```html
<img src="images/profile-photo.jpg" alt="Gulzar Ali" class="profile-image">
<img src="images/research-diagram.png" alt="Research Diagram">
```
