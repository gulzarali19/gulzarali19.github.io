# Files Directory

Store downloadable files here (CV, research papers, presentations, etc.)

## Recommended Files

- **cv.pdf** - Your curriculum vitae
- **bio.pdf** - Extended biography
- **research-paper-1.pdf** - Published papers
- **presentation.pptx** - Conference presentations

## Best Practices

1. **Naming**: Use descriptive names with dates if applicable
   - ✅ Good: `cv-2026.pdf`, `research-paper-2025.pdf`
   - ❌ Bad: `doc1.pdf`, `final-final.pdf`

2. **Organization**: Group by type
   - PDFs for academic papers
   - Documents for CV/bio
   - Archives for presentations

3. **Version Control**: 
   - Keep only current versions here
   - Archive old files elsewhere
   - Update file names with dates for clarity

## Usage in HTML

```html
<!-- Download link for CV -->
<a href="files/cv-2026.pdf" download>Download CV</a>

<!-- Link to research paper -->
<a href="files/research-paper.pdf" target="_blank">Read Paper</a>
```

## File Size Notes

Keep files reasonably sized (under 5MB recommended) for faster loading.
